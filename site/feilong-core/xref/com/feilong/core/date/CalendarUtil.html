36   * <h3>Calendar各个字段的定义:</h3>
37   * 
38   * <blockquote>
39   * 我们使用Calendar,无非是就是使用这17个字段,参见 {@link java.util.Calendar#FIELD_COUNT}
40   * 
41   * <table border="1" cellspacing="0" cellpadding="4" summary="">
42   * <tr style="background-color:#ccccff">
43   * <th align="left">字段</th>
44   * <th align="left">说明</th>
45   * </tr>
46   * <tr valign="top">
47   * <td></td>
48   * <td></td>
49   * </tr>
50   * 
51   * <tr valign="top" style="background-color:#eeeeff">
52   * <td>{@link Calendar#ERA}</td>
53   * <td>只能为0 或 1.0表示BC("before Christ",即公元前),1表示AD(拉丁语"Anno Domini",即公元)</td>
54   * </tr>
55   * 
56   * <tr valign="top">
57   * <td>{@link Calendar#YEAR}</td>
58   * <td>年</td>
59   * </tr>
60   * 
61   * <tr valign="top" style="background-color:#eeeeff">
62   * <td>{@link Calendar#MONTH}</td>
63   * <td>月 取值:可以为,JANUARY, FEBRUARY, MARCH, APRIL, MAY, JUNE, JULY, AUGUST, SEPTEMBER, OCTOBER, NOVEMBER, DECEMBER, UNDECIMBER. 其中第一个月是
64   * JANUARY,它为 0</td>
65   * </tr>
66   * 
67   * <tr valign="top">
68   * <td>{@link Calendar#WEEK_OF_YEAR}</td>
69   * <td>当前日期在本年中对应第几个星期.一年中第一个星期的值为 1</td>
70   * </tr>
71   * 
72   * <tr valign="top" style="background-color:#eeeeff">
73   * <td>{@link Calendar#WEEK_OF_MONTH}</td>
74   * <td>当前日期在本月中对应第几个星期.一个月中第一个星期的值为 1</td>
75   * </tr>
76   * 
77   * <tr valign="top">
78   * <td>{@link Calendar#DATE}</td>
79   * <td>日.一个月中第一天的值为 1</td>
80   * </tr>
81   * 
82   * <tr valign="top" style="background-color:#eeeeff">
83   * <td>{@link Calendar#DAY_OF_MONTH}</td>
84   * <td>同"DATE",表示"日"</td>
85   * </tr>
86   * 
87   * <tr valign="top">
88   * <td>{@link Calendar#DAY_OF_YEAR}</td>
89   * <td>当前日期在本年中对应第几天.一年中第一天的值为 1.</td>
90   * </tr>
91   * 
92   * <tr valign="top" style="background-color:#eeeeff">
93   * <td>{@link Calendar#DAY_OF_WEEK}</td>
94   * <td>星期几. 取值:可以为,SUNDAY、MONDAY、TUESDAY、WEDNESDAY、THURSDAY、FRIDAY 和 SATURDAY. 其中,SUNDAY为1,MONDAY为2,依次类推.</td>
95   * </tr>
96   * 
97   * <tr valign="top">
98   * <td>{@link Calendar#DAY_OF_WEEK_IN_MONTH}</td>
99   * <td>当前月中的第几个星期. 取值:DAY_OF_MONTH 1 到 7 总是对应于 DAY_OF_WEEK_IN_MONTH 1;8 到 14 总是对应于 DAY_OF_WEEK_IN_MONTH 2,依此类推. <br>
100  * DAY_OF_WEEK_IN_MONTH 指示当前月中的第几个星期.与 DAY_OF_WEEK 字段一起使用时,就可以唯一地指定某月中的某一天. <br>
101  * 与 WEEK_OF_MONTH 和 WEEK_OF_YEAR 不同,该字段的值并不 取决于 getFirstDayOfWeek() 或 getMinimalDaysInFirstWeek(). <br>
102  * DAY_OF_MONTH 1 到 7 总是对应于 DAY_OF_WEEK_IN_MONTH 1; <br>
103  * 8 到 14 总是对应于 DAY_OF_WEEK_IN_MONTH 2,依此类推. <br>
104  * DAY_OF_WEEK_IN_MONTH 0 表示 DAY_OF_WEEK_IN_MONTH 1 之前的那个星期. <br>
105  * 负值是从一个月的末尾开始逆向计数,因此,一个月的最后一个星期天被指定为 DAY_OF_WEEK = SUNDAY, DAY_OF_WEEK_IN_MONTH = -1. <br>
106  * 因为负值是逆向计数的,所以它们在月份中的对齐方式通常与正值的不同. <br>
107  * 例如,如果一个月有 31 天,那么 DAY_OF_WEEK_IN_MONTH -1 将与 DAY_OF_WEEK_IN_MONTH 5 和 DAY_OF_WEEK_IN_MONTH 4 的末尾相重叠</td>
108  * </tr>
109  * 
110  * <tr valign="top" style="background-color:#eeeeff">
111  * <td>{@link Calendar#AM_PM}</td>
112  * <td>上午 还是 下午 取值:可以是AM 或 PM.AM为0,表示上午;PM为1,表示下午.</td>
113  * </tr>
114  * 
115  * <tr valign="top">
116  * <td>{@link Calendar#HOUR}</td>
117  * <td>指示一天中的第几小时. HOUR 用于 12 小时制时钟 (0 - 11).中午和午夜用 0 表示,不用 12 表示</td>
118  * </tr>
119  * 
120  * <tr valign="top" style="background-color:#eeeeff">
121  * <td>{@link Calendar#HOUR_OF_DAY}</td>
122  * <td>指示一天中的第几小时. HOUR_OF_DAY 用于 24 小时制时钟.例如,在 10:04:15.250 PM 这一时刻,HOUR_OF_DAY 为 22</td>
123  * </tr>
124  * 
125  * <tr valign="top">
126  * <td>{@link Calendar#MINUTE}</td>
127  * <td>一小时中的第几分钟. 例如,在 10:04:15.250 PM这一时刻,MINUTE 为 4</td>
128  * </tr>
129  * 
130  * <tr valign="top" style="background-color:#eeeeff">
131  * <td>{@link Calendar#SECOND}</td>
132  * <td>一分钟中的第几秒. 例如,在 10:04:15.250 PM 这一时刻,SECOND 为 15</td>
133  * </tr>
134  * 
135  * <tr valign="top">
136  * <td>{@link Calendar#MILLISECOND}</td>
137  * <td>一秒中的第几毫秒. 例如,在 10:04:15.250 PM 这一时刻,MILLISECOND 为 250.</td>
138  * </tr>
139  * 
140  * <tr valign="top" style="background-color:#eeeeff">
141  * <td>{@link Calendar#ZONE_OFFSET}</td>
142  * <td>毫秒为单位指示距 GMT 的大致偏移量</td>
143  * </tr>
144  * 
145  * <tr valign="top">
146  * <td>{@link Calendar#DST_OFFSET}</td>
147  * <td>毫秒为单位指示夏令时的偏移量.</td>
148  * </tr>

153  * <h3>{@link Calendar#getActualMaximum(int)} VS {@link Calendar#getMaximum(int)}</h3>
154  * 
155  * <blockquote>
156  * <table border="1" cellspacing="0" cellpadding="4" summary="">
157  * <tr style="background-color:#ccccff">
158  * <th align="left">字段</th>
159  * <th align="left">说明</th>
160  * </tr>
161  * 
162  * <tr valign="top">
163  * <td>{@link Calendar#getMaximum(int)}</td>
164  * <td>获取的"字段最大值",是指在综合所有的日期,在所有这些日期中得出的"字段最大值".<br>
165  * 例如,getMaximum(Calendar.DATE)的目的是"获取‘日的最大值’".综合所有的日期,得出一个月最多有31天.因此,getMaximum(Calendar.DATE)的返回值是"31"！</td>
166  * </tr>
167  * 
168  * <tr valign="top" style="background-color:#eeeeff">
169  * <td>{@link Calendar#getActualMaximum(int)}</td>
170  * <td>获取的"当前日期时,该字段的最大值".<br>
171  * 例如,当日期为2013-09-01时,getActualMaximum(Calendar.DATE)是获取"日的最大值"是"30".当前日期是9月份,而9月只有30天.因此,getActualMaximum(Calendar.DATE)的返回值是"30"！</td>
172  * </tr>
173  * 
174  * </table>
175  * </blockquote>

192     /**
193      * 一天开始,<code>00:00:00.000</code>
194      *
195      * @param calendar
196      *            the calendar
197      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}
198      * @see Calendar#set(int, int)
199      * @see Calendar#HOUR_OF_DAY
200      * @see Calendar#MINUTE
201      * @see Calendar#SECOND
202      * @see Calendar#MILLISECOND
203      * @see org.apache.commons.lang3.time.DateUtils#truncate(Calendar, int)
204      * @since 1.3.0
205      */
206     static Calendar resetDayBegin(Calendar calendar){
207         Validate.notNull(calendar, "calendar can't be null!");
208         calendar.set(HOUR_OF_DAY, 0);
209         calendar.set(MINUTE, 0);
210         calendar.set(SECOND, 0);
211         calendar.set(MILLISECOND, 0);
212         return calendar;
213     }
214 
215     /**
216      * 一天结束,最后的时间 <code>23:59:59.999</code>
217      *
218      * @param calendar
219      *            the calendar
220      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}
221      * @see Calendar#set(int, int)
222      * @see Calendar#HOUR_OF_DAY
223      * @see Calendar#MINUTE
224      * @see Calendar#SECOND
225      * @see Calendar#MILLISECOND
226      * @since 1.3.0
227      */
228     static Calendar resetDayEnd(Calendar calendar){
229         Validate.notNull(calendar, "calendar can't be null!");
230         calendar.set(HOUR_OF_DAY, 23);
231         calendar.set(MINUTE, 59);
232         calendar.set(SECOND, 59);
233         calendar.set(MILLISECOND, 999);
234         return calendar;
235     }
236 
237     /**
238      * 一年结束,最后的时间 <code>12月31号 23:59:59.999</code>
239      *
240      * @param calendar
241      *            the calendar
242      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}
243      * @since 1.3.0
244      * @see #resetDayEnd(Calendar)
245      */
246     static Calendar resetYearEnd(Calendar calendar){
247         Validate.notNull(calendar, "calendar can't be null!");
248         calendar.set(MONTH, DECEMBER);
249         calendar.set(DAY_OF_MONTH, 31);
250         return resetDayEnd(calendar);
251     }

257     /**
258      * 将 {@link Calendar} 转成 {@link Date}.
259      * 
260      * @param calendar
261      *            calendar
262      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}
263      * @see java.util.Calendar#getTime()
264      */
265     static Date toDate(Calendar calendar){
266         Validate.notNull(calendar, "calendar can't be null!");
267         return calendar.getTime();
268     }
269 
270     /**
271      * 将{@link Calendar}转成{@link String}.
272      * 
273      * @param calendar
274      *            calendar
275      * @param datePattern
276      *            日期pattern {@link DatePattern}
277      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}
278      * @see com.feilong.core.date.DateUtil#toString(Date, String)
279      */
280     static String toString(Calendar calendar,String datePattern){
281         Date date = toDate(calendar);
282         return DateUtil.toString(date, datePattern);
283     }

287     /**
288      * 获得日历字段值.
289      *
290      * @param date
291      *            date
292      * @param field
293      *            Calendar字段:<br>
294      *            月份:{@link Calendar#MONTH}(真实值需要加1处理),<br>
295      *            日:{@link Calendar#DAY_OF_MONTH},<br>
296      *            年份:{@link Calendar#YEAR}<br>
297      *            ...
298      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}<br>
299      * @see #getFieldValue(Calendar, int)
300      * @since 1.3.0
301      */
302     static int getFieldValue(Date date,int field){
303         Calendar calendar = DateUtil.toCalendar(date);
304         return getFieldValue(calendar, field);
305     }
306 
307     /**
308      * 获得日历字段值.
309      *
310      * @param calendar
311      *            the calendar
312      * @param field
313      *            Calendar字段:<br>
314      *            月份:{@link Calendar#MONTH}(真实值需要加1处理),<br>
315      *            日:{@link Calendar#DAY_OF_MONTH},<br>
316      *            年份:{@link Calendar#YEAR}<br>
317      *            ...
318      * @return 如果 <code>calendar</code> 是null,抛出 {@link NullPointerException}<br>
319      * @see java.util.Calendar#get(int)
320      * @since 1.3.0
321      */
322     static int getFieldValue(Calendar calendar,int field){
323         Validate.notNull(calendar, "calendar can't be null!");
324         return calendar.get(field);
325     }
